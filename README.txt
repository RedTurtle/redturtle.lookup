Introduction
============

This product is done for the not common situation where you need to manage multiple Plone sites in the same
Data.fs.

How to use
==========

It add to your Zope root (*not* to your Plone site root) two new additional views:

``@@lookup.site``
    Simply  display an *url-to-site sitename* list, with an entry for every site.
``@@lookup``
    Display a detailed table (with accordion support) that display a lot of information and links to more
    useful Plone features, for every site.
